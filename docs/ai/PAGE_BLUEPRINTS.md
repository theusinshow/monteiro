## Home Blueprint

### Objective
Establish authority instantly, curate the best work, and open a quiet path to contact.

### Hierarchy
1. Brand presence + tone (hero)
2. Curated projects (proof)
3. Philosophy (manifesto)
4. Quiet contact invitation

### Low-fidelity structure
```text
[Header — wordmark · Projetos / Estúdio / Contato]
[Hero — hairlines draw in · studio name + short statement · hero image/featured project]
[Selected Projects — 3–5 items, asymmetric grid, hairline separators]
[Manifesto — 1–2 line philosophy, large whitespace]
[Context numbers — optional: anos · projetos · tipos (placeholder)]
[Quiet CTA — "Vamos conversar" → /contato]
[Footer-index — nav grid · contato · social · hairline frame]
```

### Responsive behavior
Desktop: exposed multi-column grid, asymmetric project placement. Tablet: 2-column, reduced margins. Mobile: single column, stacked projects, hairlines become full-width dividers; hero image prioritized for LCP.
