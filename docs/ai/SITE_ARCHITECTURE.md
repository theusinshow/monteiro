## Global Navigation Rules
- Minimal header: studio wordmark (left) + nav (Projetos, Estúdio, Contato) + discreet contact affordance.
- Header stays light/transparent over media, gains a hairline divider on scroll.
- A quiet, persistent contact CTA appears site-wide (footer always; inline near end of key pages).
- Footer acts as an index: navigation grid + contact + social, framed by hairlines.
- Active states and hover reveal hairlines, not heavy fills.

## Page Architecture

### Home
- Objective: Establish authority in seconds; route to the portfolio; open a quiet path to contact.
- Sections: Hero editorial (hairlines draw-in) · Selected projects (asymmetric grid) · Manifesto (1–2 lines) · Context numbers (optional, placeholder) · Quiet contact CTA · Footer-index.
- Primary CTA: Ver projetos (secondary: Contato).
- Notes: Photography-forward; text whispers. Hairlines define margins and section breaks.

### Projects (index) — `/projetos`
- Objective: Present curated work as the central proof.
- Sections: Page title + intro line · Filter by type (Residencial / Comercial) · Editorial list/grid of project cards (image, name, year, location) · Footer.
- Primary CTA: Open a project.
- Notes: Hover reveals hairline + metadata.

### Project (detail) — `/projetos/[slug]`
- Objective: Deepen desire through one project's narrative.
- Sections: Full-bleed cover · Technical sheet in grid (local, ano, área, tipo) · Editorial vertical gallery with breathing room · Next project link · Footer.
- Primary CTA: Próximo projeto / Contato.
- Notes: Consistent image ratios; progressive loading.

### Studio / About — `/estudio`
- Objective: Build trust in the person and philosophy.
- Sections: Portrait · Philosophy statement · Approach/process · Recognition (optional, placeholder) · Contact CTA · Footer.
- Primary CTA: Contato.
- Notes: Placeholder copy until provided.

### Contact — `/contato`
- Objective: Convert with zero friction.
- Sections: Short invitation · Minimal form (nome, e-mail/telefone, mensagem) · WhatsApp direct + email · Footer.
- Primary CTA: Enviar / Falar no WhatsApp.

## Conversion Path
Authority (Home/Projects) → Desire (Project detail) → Trust (Studio) → Action (Contact). Discreet CTA always reachable; never interrupts the editorial flow.

## SEO / Accessibility / Performance Requirements
- One coherent H1 per page; semantic headings; descriptive titles/meta.
- Per-project metadata for `/projetos/[slug]` (Open Graph image = cover).
- Keyboard navigable; visible focus; alt text for meaningful images; `prefers-reduced-motion` honored.
- Optimize LCP (hero image), avoid CLS (reserved media ratios), protect INP.

## Asset Guidelines
- Use optimized images (`next/image`, AVIF/WebP, responsive sizes).
- Prefer SVG for logo and icons.
- Avoid embedded text inside images when content should be indexed or translated.
- Define meaningful alt text.
- Keep video and 3D assets conditional and performance-aware.
