# Design System Foundations

> Generated by Coded by M — Site Architecture System. Source of truth: `/docs/ai/DISCOVERY.md` and `/docs/ai/DECISIONS_LOG.md`.

## Visual Direction
Minimalist, sophisticated, architectural, editorial, silent, premium — with generous breathing room and zero template feel. **Signature system: an exposed hairline (1px) grid** that behaves like an architectural drawing — it marks margins, exposes columns, sets a baseline rhythm and separates sections. Lines are structure, never ornament. Photography is the protagonist; UI recedes.

## Color Foundations
Warm near-monochrome; chromatic richness still comes mainly from project photography. One committed secondary accent (editorial brick) relieves the monochrome — used sparingly, never decoratively (see DEC-008).
- `--color-paper` Off-white background — `#f3f1ec`
- `--color-paper-deep` Recessed surface — `#eceae3`
- `--color-ink` Primary text / lines — `#1a1714`
- `--color-graphite` Secondary text — `#6c685f`
- `--color-stone` Warm taupe neutral — `#9a8c76` (sparingly)
- `--color-line` / `--color-line-strong` Hairlines — `--ink` at 14% / 28%
- `--color-accent` **Secondary — editorial brick** `oklch(0.47 0.155 32)` (≈ `#9f2c19`). AA-safe on paper (6.53:1). Reserved for: signature display italic, wordmark mark, index/folio numerals, active filter state, link-hover underline, focus ring, selection. Keep under ~10% of any view.
Maintain AA contrast for text; never let the accent compete with photography or the hairline grid.

## Typography
- **Display (serif, editorial):** large headings and the studio statement. Candidate vibe: a high-contrast modern serif. (placeholder family)
- **Text (grotesque / neo-grotesque):** body, labels, metadata, nav. (placeholder family)
- Wide type scale, controlled tracking, generous leading. Numerals tabular for technical sheets.
- Load via `next/font` for performance and no layout shift.

## Spacing Scale
Base 8px rhythm, tied to the baseline grid:
`4 · 8 · 12 · 16 · 24 · 32 · 48 · 64 · 96 · 128 · 160`
Section padding is intentionally large (96–160) to create silence.

## Containers and Grid
- Max content width ~1280–1440px with wide outer margins.
- **12-column grid with visible hairline gutters/columns** on larger breakpoints.
- Baseline grid governs vertical rhythm.
- Asymmetry is allowed and encouraged for editorial tension (projects offset within the grid).

## Radius and Shadows
- Radius: 0 (sharp, architectural). Optional 2px max only if ever needed.
- Shadows: avoid. Depth comes from layout, hairlines and imagery, not shadow.

## Essential Components
- Navbar (wordmark + minimal nav + hairline-on-scroll)
- Footer (index grid + contact, hairline-framed)
- Buttons (text/ghost style with hairline underline or border; no heavy fills)
- Links (hairline underline reveal on hover)
- Containers / grid wrappers (expose columns)
- Section headers (label + large display title)
- Project card (image + name + year + location; hover hairline)
- Technical sheet (label/value grid)
- Form fields (underline-only, minimal)
- Tags/badges (type filter: Residencial / Comercial) — text + hairline, no pill fill

## Responsive Principles
Desktop exposes the full grid and asymmetry; tablet simplifies to 2 columns; mobile collapses to a single column where hairlines become full-width dividers. Preserve whitespace ratios; never crowd. Maintain image aspect ratios across breakpoints.

## Asset Guidelines
- `next/image` with AVIF/WebP, responsive `sizes`, blur placeholder.
- SVG for wordmark/icons; 1px strokes for line motifs.
- Consistent, intentional crop ratios for project imagery.

## Reference Principles
Think architectural monograph / editorial print: high-end studio portfolios, design annuals, gallery sites. Restraint over decoration; precision over flourish. Every line should feel measured.
