## Visual Direction
Minimalist, sophisticated, architectural, editorial, silent, premium — with generous breathing room and zero template feel. **Photography leads the identity** ("monografia / prancha montada"): large plates on the dark ground, with corner registration marks framing each image like a drawing mounted on a sheet. The only line system is **quiet horizontal hairline rules contained within the content margin** (they begin and end at the content edge, never edge-to-edge) that separate sections. Alignment stays disciplined via the content band but is no longer painted as an exposed grid. Home opens with an editorial cover (statement left, tall photograph right). Lines are structure, never ornament; the UI recedes so the photography reads. (See DEC-015; supersedes the exposed-grid signature of DEC-005.)

## Color Foundations
**Dark "Drafting" direction (DEC-014).** Warm near-monochrome, inverted: the surface is a dark drafting board and the hairline grid reads as a luminous line drawn on it. Chromatic richness still comes mainly from project photography, which now reads as lit plates on a gallery wall. The role-based token names persist (`paper` = ground, `ink` = the line/text) even though the metaphor inverts. One committed secondary accent (editorial brick) relieves the monochrome — used sparingly, never decoratively (see DEC-008). All OKLCH; neutrals tinted to the brand hue; no pure `#000`/`#fff`.
- `--color-paper` Dark warm ground — `oklch(0.205 0.009 62)`
- `--color-paper-deep` Recessed surface (frames) — `oklch(0.172 0.009 62)`
- `--color-ink` Light primary text / lines — `oklch(0.925 0.013 83)`
- `--color-graphite` Secondary text — `oklch(0.66 0.012 75)` (AA on ground)
- `--color-stone` Warm taupe neutral — `oklch(0.62 0.025 75)` (sparingly)
- `--color-line` / `--color-line-strong` Hairlines — `--ink` at 16% / 30% (a touch more alpha than the light theme so the grid reads on the dark ground)
- `--color-accent` **Secondary — editorial brick** `oklch(0.66 0.15 40)`. Brightened from the light-theme `0.47` so the small folio numerals clear AA 4.5:1 on the dark paper. Reserved for: signature display italic, wordmark mark, index/folio numerals, active filter state, link-hover underline, focus ring, selection. Keep under ~10% of any view.
Maintain AA contrast for text; never let the accent compete with photography or the hairline grid. `color-scheme: dark`.

## Typography
- **Display (serif, editorial):** large headings and the studio statement. Candidate vibe: a high-contrast modern serif. (placeholder family)
- **Text (grotesque / neo-grotesque):** body, labels, metadata, nav. (placeholder family)
- Wide type scale, controlled tracking, generous leading. Numerals tabular for technical sheets.
- Load via `next/font` for performance and no layout shift.

## Spacing Scale
Base 8px rhythm, tied to the baseline grid:
`4 · 8 · 12 · 16 · 24 · 32 · 48 · 64 · 96 · 128 · 160`
Section padding is intentionally large (96–160) to create silence.

## Containers and Grid
- Max content width ~1280–1440px (`--grid-max`) with wide outer margins (`--gutter`).
- Alignment via a centered content band (`Container`) + CSS grid/flex — **disciplined but not painted** (no persistent vertical hairlines, no intersection crosshairs).
- Horizontal section rules (`Rule`) share the band width, so lines and content end at the same content margin.
- Asymmetry is encouraged for editorial tension (projects offset via vertical staggers, varied image sizes).

## Radius and Shadows
- Radius: 0 (sharp, architectural). Optional 2px max only if ever needed.
- Shadows: avoid. Depth comes from layout, hairlines and imagery, not shadow.

## Essential Components
- Navbar (wordmark + minimal nav + hairline-on-scroll)
- Footer (index grid + contact, hairline-framed)
- Buttons (text/ghost style with hairline underline or border; no heavy fills)
- Links (hairline underline reveal on hover)
- Containers / grid wrappers (expose columns)
- Section headers (label + large display title)
- Project card (image + name + year + location; hover hairline)
- Technical sheet (label/value grid)
- Form fields (underline-only, minimal)
- Tags/badges (type filter: Residencial / Comercial) — text + hairline, no pill fill

## Responsive Principles
Desktop exposes the full grid and asymmetry; tablet simplifies to 2 columns; mobile collapses to a single column where hairlines become full-width dividers. Preserve whitespace ratios; never crowd. Maintain image aspect ratios across breakpoints.

## Asset Guidelines
- `next/image` with AVIF/WebP, responsive `sizes`, blur placeholder.
- SVG for wordmark/icons; 1px strokes for line motifs.
- Consistent, intentional crop ratios for project imagery.

## Reference Principles
Think architectural monograph / editorial print: high-end studio portfolios, design annuals, gallery sites. Restraint over decoration; precision over flourish. Every line should feel measured.
